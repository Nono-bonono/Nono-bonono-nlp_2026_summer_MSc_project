# NLP Course, final project

---- This project is currently under construction ----

This is a project for my NLP course at Birkbeck College. The project divides into 3 parts.

The first deals with basic tokenization and analysis of written text using the Flesch-Kincaid ease of reading score.
In addition we use spaCy to look at PMI for certain words etc. The basics of NLP.

The second part is focused on traing and testing ML classifiers on text. The texts are political speeches
and the goal is for the model to classify to which party the speech belongs. We use 2 different classifiers (RandomForest and SVMs) and then play around with building
a custom tokenizer.

Third part is the same task as part 2 but using off the shelf LLMs. For this part I used meta-llama/llama-3.3-70b-instruct accessed via paid access from OpenRouter.
