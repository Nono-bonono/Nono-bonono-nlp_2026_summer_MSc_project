# NLP Course, final project

This project divides into 3 parts.
The first deals with basic tokenization and analysis of written text using the Flesch-Kincaid ease of reading score.
Another primary objective is basic data wrangling and processing in terms of pandas and pickling.

The second part is focused on traing and testing ML classifiers on text. The texts are political speeches
and the goal is for the model to classify to which party the speech belongs.

Third part is the same task as part 2 but using off the shelf LLMs
